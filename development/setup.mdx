# Development Setup

Hướng dẫn cài đặt môi trường để phát triển Appota HRM.

## Yêu cầu

## Cài đặt ban đầu

<Steps>
  <Step title="Clone repository">
    ```bash
    git clone https://github.com/Agentlize/HRM.git
    cd HRM/hrm-appota
    ```
  </Step>
  <Step title="Cài đặt dependencies">
    ```bash
    pnpm install
    ```

    Hoặc dùng npm:

    ```bash
    npm install
    ```
  </Step>
  <Step title="Tạo file .env">
    Tạo file `.env` ở root `hrm-appota/`:

    ```env
    VITE_SUPABASE_URL=https://your-project.supabase.co
    VITE_SUPABASE_ANON_KEY=your-anon-key
    ```

    <Note>
      Hệ thống chạy được với mock data mà không cần Supabase. Biến môi trường chỉ bắt buộc khi muốn kết nối database thật.
    </Note>
  </Step>
  <Step title="Chạy kiểm tra">
    ```bash
    npm run lint
    ```

    Lệnh này chạy:

    - **TypeScript check** (`tsgo -p tsconfig.check.json`)
    - **Biome lint** (`npx biome lint`)
    - **Custom checks** (`.rules/check.sh`)
    - **Tailwind validation**
    - **Test build** (`.rules/testBuild.sh`)
  </Step>
</Steps>

## Scripts

| Script | Mô tả |
| --- | --- |
| `npm run dev` | Vite dev server (bị disable trong sandbox) |
| `npm run build` | Production build |
| `npm run start` | Preview production build |
| `npm run lint` | TypeScript \+ Biome \+ custom checks |

## IDE Setup

### VSCode

Khuyến nghị extensions:

- **Biome** (`biomejs.biome`)
- **Tailwind CSS IntelliSense** (`bradlc.vscode-tailwindcss`)
- **ESLint** (legacy, dùng để xem rule từ Biome)
- **TypeScript Vue Plugin** (Volar) (`vue.volar`)
- **Error Lens** (`usernamehw.errorlens`)
- **Auto Rename Tag** (`formulahendry.auto-rename-tag`)

## Debug

### Browser DevTools

- **React DevTools** — xem component tree, props, state
- **Redux DevTools** — không cần vì dùng Context
- **Network tab** — kiểm tra fetch requests (khi có backend)

### Common Issues

| Vấn đề | Giải pháp |
| --- | --- |
| `npm run dev` không chạy | Trong môi trường sandbox, dùng `npm run lint` thay thế |
| TypeScript error | Chạy `npx tsc --noEmit` để xem chi tiết |
| Tailwind class không hoạt động | Kiểm tra `content` paths trong `tailwind.config.js` |
| Dark mode không gắn vào `<html>` | Kiểm tra `ThemeContext` có toggle `dark` class |

## Tích hợp Supabase

<Steps>
  <Step title="Tạo project trên Supabase">
    Truy cập [supabase.com](https://supabase.com) → New project.
  </Step>
  <Step title="Copy URL và Anon Key">
    Settings → API → Copy `URL` và `anon public key`.
  </Step>
  <Step title="Paste vào .env">
    ```env
    VITE_SUPABASE_URL=https://xxx.supabase.co
    VITE_SUPABASE_ANON_KEY=eyJ...
    ```
  </Step>
  <Step title="Khởi tạo schema">
    Sử dụng schema từ `src/db/supabase.ts` hoặc tạo migration mới.
  </Step>
</Steps>
