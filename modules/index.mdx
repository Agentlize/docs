# Modules

Appota HRM gồm **18 modules chính**, được nhóm thành 5 nhóm nghiệp vụ. Mỗi module có một route riêng và được khai báo trong `src/routes.tsx`.

| `jdData.ts` / `jdPoolData.ts` / `jdRequestData.ts` | JD | 15\+ JD |
| `cvPipelineData.ts` / `cvPoolData.ts` | CV | 30\+ CV |
| `interviewData.ts` | Interviews | 5-6 buổi phỏng vấn |
| `recruitmentOrderData.ts` | Recruitment Orders | 10\+ orders |
| `permissionsData.ts` | Roles, permissions, policies | 9 roles, 24 permissions |
| `auditData.ts` | Audit log | 150-200 entries |
