# Staff Detail

**Route:** `/staff/:id`

Trang chi tiết nhân viên hiển thị **đầy đủ thông tin cá nhân**, activity timeline và **11 tab chức năng** phục vụ quản lý nhân sự, đào tạo và phát triển.

## Activity Input

Khi nhập activity mới vào input box cuối Left Panel:

```typescript title="HRMContext.tsx"
addActivity(staffId, {
  type: 'note',
  title: 'Tham gia meeting',
  description: 'Họp team Q2',
  timestamp: new Date(),
});
```

Activity mới xuất hiện ngay đầu timeline trong tab **Activity**.

## Quyền truy cập

<Warning>
  - **Employee** chỉ xem được profile của **chính mình** (permission boundary).
  - **Department Head** xem được profile trong **phòng ban mình**.
  - **HR Manager / Super Admin** xem được **toàn bộ**.
</Warning>

## API
