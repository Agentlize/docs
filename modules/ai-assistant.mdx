# AI Assistant

**Route:** `/ai-chat`

Trợ lý AI hỗ trợ HR trong công việc hàng ngày. Sử dụng **SSE streaming** để trả lời theo thời gian thực.

## Hook useAIChat

```typescript title="src/hooks/useAIChat.ts"
import { useAIChat } from '@/hooks/useAIChat';

function HRChatPage() {
  const { messages, isStreaming, send, stop } = useAIChat({
    endpoint: '/api/ai/chat',
    onFinish: (msg) => console.log('done:', msg),
  });

  return (
    <div>
      {messages.map(m => (
        <Message key={m.id} role={m.role} content={m.content} />
      ))}
      <ChatInput onSubmit={send} onStop={stop} disabled={isStreaming} />
    </div>
  );
}
```

## SSE Client

```typescript title="src/lib/sse.ts"
const stream = new SSE('/api/ai/chat', {
  method: 'POST',
  body: JSON.stringify({ message }),
});

stream.addEventListener('message', (event) => {
  // Append token to current message
});
```

## Tính năng

- **Streaming response** — token xuất hiện theo thời gian thực
- **Stop button** — dừng stream giữa chừng
- **Message history** — lưu lịch sử chat
- **Markdown rendering** — hỗ trợ format câu trả lời

## Use cases

- Hỏi về quy trình HR
- Tra cứu chính sách công ty
- Hỗ trợ viết JD
- Phân tích dữ liệu nhân sự
