# Audit Trail

**Route:** `/audit`

Theo dõi tất cả các thao tác thay đổi trong hệ thống với **AI đánh giá rủi ro** cho từng entry.

## Audit Entry

Mỗi entry chứa:

| Trường | Mô tả |
| --- | --- |
| Timestamp | Thời điểm thực hiện |
| Actor | User \+ avatar |
| Action Type | CREATE / UPDATE / DELETE / APPROVE / REJECT / STATUS\_CHANGE |
| Module | Staff, Interview, HR Review, v.v. |
| Entity | Staff ID, Interview ID, CV ID, JD ID, Order ID, Course ID |
| Before | Giá trị trước |
| After | Giá trị sau |
| Risk Level | Low / Medium / High / Critical |

## AI Risk Assessment

Mỗi entry được AI đánh giá:

```typescript
type RiskLevel = 'Low' | 'Medium' | 'High' | 'Critical';

interface RiskAssessment {
  level: RiskLevel;
  explanation: string;
  recommendations: string[];
}
```

### Phân loại Risk

| Risk Level | Ví dụ |
| --- | --- |
| **Low** | View Staff, View CV, View Course |
| **Medium** | Edit Staff, Edit CV, Edit Course |
| **High** | Delete Staff, Reject Onboarding, Delete CV, Delete Course |
| **Critical** | Delete nhiều Staff, Thay đổi permission Super Admin, Delete nhiều CV |

### Ví dụ assessment

```text
[2026-05-21 09:00] HR Trần Thị B
Action: HR_REVIEW → APPROVED
Module: Recruitment Orders
Entity: Order #1234
Risk Level: Low
Explanation: "Normal status transition trong recruitment workflow."
Recommendation: ["No action required"]
```
